## 📋 10 分标准检查清单（每页都要过）

### 必过项（Hard Requirements）
- [ ] 无 console error
- [ ] 无 happy talk 文案（"欢迎使用""感谢使用"等）
- [ ] 所有 body 文本 >= 16px（mobile）
- [ ] 对比度 >= 4.5:1（WCAG AA）
- [ ] 无 AI slop（无紫渐层、无 3 列卡片阵列、无 emoji 作设计、无 cookie-cutter 节奏）
- [ ] 所有 CSS 用变量（无硬编码 hex）
- [ ] 所有颜色用 CSS var()
- [ ] 所有间距遵循 8px 基数
- [ ] CTA 按钮有 hover/active 反馈
- [ ] 动画只用 transform/opacity
- [ ] 所有表单 input 都有清晰 label
- [ ] focus-visible 状态存在
- [ ] 响应式 375/768/1024 都 OK
- [ ] 无水平滚动

### 文案质量项（Content Requirements）
- [ ] 故事叙述遵循"先坏后救"（先说问题，再说解决方案）
- [ ] 建议具体（包含月份/方向/颜色/物品，不是虚假承诺）
- [ ] 按钮文案明确（"推算合婚"不是"继续"）
- [ ] 没有句子超过 30 字（可扫描性）
- [ ] 用**粗体**标记关键建议

### 交互质量项（Interaction Requirements）
- [ ] 所有异步操作都有 loading UI（最少 2.5s）
- [ ] 错误提示具体友好
- [ ] 成功反馈有 animation
- [ ] touch target >= 44px
- [ ] 点击所有元素都有反馈（不是无声操作）
